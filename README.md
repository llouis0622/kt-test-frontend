# 부산시 청년 창업 챗봇 프론트엔드

이 프로젝트는 부산시 청년 창업 지원을 위한 챗봇 웹 프론트엔드입니다. React, TypeScript, Material-UI(MUI) 기반으로 제작되었습니다.

## 실행 방법

1. 의존성 설치

```
npm install
```

2. 개발 서버 실행

```
npm start
```

3. 브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

---

## 주요 기능
- 카테고리/세부 항목 선택
- ChatGPT 스타일의 질문/답변 UI
- 추후 백엔드 연동 가능 (현재는 Mock 데이터 사용)
